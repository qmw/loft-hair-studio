# Loft Hair Studio ✂️

Strona-wizytówka fikcyjnego autorskiego salonu fryzjerskiego — projekt do **portfolio stron internetowych dla firm**.

Ciemny, luksusowo-edytorski minimalizm: niemal czarne tło, kostny biały tekst, piaskowy akcent, ostre kanty i wysokokontrastowa typografia modowa. Jedna responsywna strona (one-page) z animacjami i interaktywnym suwakiem metamorfoz „przed / po".

## ✨ Sekcje

- **Hero** — asymetryczny układ z animowanym wjazdem typografii
- **Marquee** — przewijający się pasek usług
- **O studiu** — opis + liczniki
- **Usługi** — lista z numeracją i hoverem
- **Metamorfozy** — interaktywny suwak **przed / po**
- **Cennik** — lista cen usług
- **Zespół** — portrety stylistów (kolor na hover)
- **Rezerwacja** — formularz wizyty online + mapa + godziny i kontakt

## 🛠️ Stack

- [Astro 5](https://astro.build/)
- [Tailwind CSS 4](https://tailwindcss.com/)
- Typografia: Bodoni Moda · Archivo (Google Fonts)

## 🚀 Uruchomienie

```bash
npm install
npm run dev      # serwer deweloperski → http://localhost:4321
npm run build    # build produkcyjny do ./dist
npm run preview  # podgląd builda
```

## ☁️ Hosting

Projekt statyczny — gotowy do darmowego hostingu na [Vercel](https://vercel.com/) (import repozytorium → framework wykrywany automatycznie jako Astro).
